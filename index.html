WideOut X

Workflow automation. Click the assistant in the bottom-right to talk to the genie.

How to embed

<script src="https://wideoutx.github.io/wox-chatbot/wox_chat_widget.js"></script>
<script>
  WoxChat.init({
    endpoint: 'https://apim.workato.com/.../chat-api',
    apiToken: 'YOUR_TOKEN',
    title:    'WideOut X Assistant',
    subtitle: 'Workflow automation guide',
    welcomeMessage: 'Hello — welcome to WideOut X.\nWhat would you like to explore?',
    welcomeQuickReplies: [
      { label: 'Automate a Workflow', icon: 'flow', message: 'I want to automate a workflow' },
      { label: 'Services',            icon: 'work', message: 'Tell me about your services' },
      { label: 'Pricing',             icon: 'tag',  message: 'What are your pricing plans?' },
      { label: 'Careers',             icon: 'case', message: 'Are you hiring?' }
    ],
    footerTagline: 'Map your Manual Workflow and see the path to an Automated Workflow',
    accentColor: '#22d3ee'
  });
</script>

⚠ The API token is visible in browser source. For production, proxy through your own backend.

Dynamic quick replies from the genie

To let the genie suggest follow-up buttons on any turn, update your recipe's API response to include a quick_replies array: 

{
  "reply": "Sure! What plan size are you looking for?",
  "conversation_id": "...",
  "quick_replies": ["Starter", "Team", "Enterprise"]
}

The widget will render them as chips under that message. No widget code changes needed.
